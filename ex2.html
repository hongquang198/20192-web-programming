Student number is a number containing 8 digits in HUST (ex. 20163316) which is used to identify students.
The first 4 digits specifies the year of entrance.

Birth name is a name given to a person when born

Nick name is a casual name given to you and is used by close friends

Version ID is a number specifying the version of an application

Module code is a string of numbers and letters used to specify a number of courses available
